# 多尺度双轨制煤炭价格智能预测系统

## 项目简介
本项目是基于中国双轨制定价体系下的煤炭价格预测系统，使用多尺度分析和机器学习技术进行价格预测。

## 功能特点
- 多尺度煤炭价格预测（日度、月度、年度）
- 双轨制定价系统展示（市场价、长协价、基准价）
- 交互式数据可视化图表
- 响应式设计，支持移动端和桌面端
